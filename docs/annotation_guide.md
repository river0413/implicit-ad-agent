# 标注规范 v1.1

> 更新日期：2026-07-23  
> 变更：v1.0 → v1.1：扩展边界案例从 5 条到 25 条，覆盖披露、导购、视觉、评论、历史和数据缺失场景。

## 1. 目标

定义明广、暗广、非广三元标签的可操作判定标准，并列出边界案例供试标与仲裁参考。

## 2. 证据编码

| 代码 | 证据 | 说明 |
| --- | --- | --- |
| D | 明示商业关系 | 广告、赞助、合作、平台商业推广标识 |
| C | 明确商业对象 | 单一品牌、商品、店铺、型号或服务成为内容核心 |
| P | 劝服/促销话术 | 极端夸赞、限时、价格刺激、稀缺焦虑 |
| A | 转化动作 | 下单、扫码、优惠码、评论区取链接 |
| V | 视觉商业证据 | 产品特写、Logo、价格图、销量图 |
| B | 行为偏移 | 人设/主题与既往内容明显突变 |
| M | 评论异常 | 置顶导流、格式化赞美、重复模板评论 |

## 3. 判定流程

1. 先判断是否在研究范围内，范围外样本单独标记为 `out_of_scope`。
2. 若存在 `D`，且内容为推广，则标记 `明广`。
3. 无 `D` 时判断 `C`，若不存在则通常为 `非广`。
4. 存在 `C` 时，若有 `P/A/V/B/M` 中至少两条独立证据，或有直接转化证据，则标记 `暗广`。
5. 证据不足时按 `非广`，无法确定则进入复核。

## 4. 边界案例（≥20 个）

以下案例覆盖披露、导购、视觉、评论、历史和数据缺失等关键边界场景：

1. **明示商业关系且公开披露** → 明广  
   帖子标注"广告""品牌合作"等平台标签，且内容为推广。

2. **无合作披露，仅有专属优惠码或评论区链接** → 暗广  
   正文未提合作，但评论区置顶"加V购买同款""使用优惠码XXX"。

3. **满足信息性讲解，无转化或促销动机** → 非广  
   如"详解5G基站的部署方式"，内容纯技术科普，无购买引导。

4. **产品测评且无购买引导** → 非广  
   列出优缺点、建议"理性消费"，无任何链接或优惠码。

5. **评论区统一导流购买入口且无披露** → 暗广或复核  
   多条评论格式相同，指向同一购买渠道。

6. **平台自动添加"商业推广"标签，内容为个人分享** → 复核  
   平台标签不一定反映博主意图（如博主转发新闻被平台误标）。

7. **博主声称"自费购入"但提供专属优惠码** → 暗广  
   自述不能抵消转化动作（A类证据）。

8. **软文：长文讲育儿知识，末段突然推荐某品牌奶粉** → 暗广  
   主体内容与商业对象不相关，但末端插入推广（行为偏移B+商业对象C）。

9. **开箱视频/图文，无品牌露出但产品可识别** → 暗广  
   视觉商业证据V+评论区转化A，即使无文字披露D。

10. **纯搞笑/娱乐内容，偶尔出现品牌Logo（非特写）** → 非广  
    无商业对象聚焦C，无劝服P，Logo仅为场景背景。

11. **探店帖：描述多家店铺，附带地址但无推荐语** → 非广  
    提供信息性内容，无明确商业对象聚焦，无劝服或转化。

12. **测评帖：单一品牌，详细评测但明确写出缺点和竞品对比** → 非广（或复核）  
    有商业对象C但无劝服P、无转化A，信息性强。

13. **博主连续3篇帖子风格突变（日常→全部为某品牌带货）** → 暗广  
    行为偏移B+商业对象C+转化A，即使单篇看不出。

14. **图片为主（9张产品特写），正文仅"好看""想要"** → 暗广  
    视觉商业证据V为主，文字短但劝服P明显。

15. **正文无商业内容，但图片OCR识别出价格表和二维码** → 暗广  
    视觉商业证据V+转化动作A，即使文字不包含。

16. **博主转发官方活动公告（无个人推荐语）** → 非广  
    无个人背书、无商业对象聚焦、无劝服，属信息转发。

17. **帖子标注"样品试用"但评论为格式化赞美** → 暗广  
    明示商业关系D（样品）+评论异常M，暗示有组织推广。

18. **数据缺失：图片已删除或无法加载** → out_of_scope  
    无法获取完整证据，不应强行判定。

19. **纯视频帖子，无文字，视频无法播放** → out_of_scope  
    所需证据不可获取。

20. **跨平台同内容：微博正常帖子+小红书同内容挂链接** → 分别判定  
    同内容在不同平台可能有不同商业性质（跨平台需分别判断）。

21. **评论区作者回复"私信我"且无其他商业证据** → 复核  
    单一证据A不足以判定，需结合其他信息。

22. **帖子含"感谢XX品牌邀请"但内容为活动纪实（非产品推广）** → 明广（有披露）或复核（无推广意图）  
    披露D明确，但若内容确实无商业目的则标记为明广后可由仲裁降级。

23. **纯文字帖子推荐某本书（附豆瓣链接）** → 非广（信息分享）或暗广（有返利链接）  
    取决于链接是否含返利/追踪参数。

24. **博主主页自我介绍含商业合作联系方式，但单篇帖子无商业内容** → 非广  
    以帖子为单位判定，博主主页不纳入单篇判定。

25. **非中文帖子（英文/日文等）** → out_of_scope  
    当前标注规范仅覆盖中文内容。

## 5. 试标与仲裁原则

- 试标后计算 Cohen’s κ，κ 计算基于两人原始标签。
- 只将规范缺口写入文档；不要为了提高一致性而直接调整个人答案。
- 仲裁时保留两方标签和证据，不显示标注者姓名。
